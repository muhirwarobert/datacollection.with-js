# DataCollection !['coverage'](http://img.shields.io/badge/coverage-100%25-brightgreen.svg)

To install simply use

```
$ npm install data-collection
```

And

```
var DataCollection = require('data-collection');
```

You're ready to go!

# Tests

Run tests with

```
$ npm test
```

in the ```node``` directory.

Voila, 100% coverage. :)
